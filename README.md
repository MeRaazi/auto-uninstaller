# Auto Uninstaller [KSU]
**A useless kernelsu module to automatically uninstall bloacklisted apps.**

![Auto-Uninstaller Screenshot](https://raw.githubusercontent.com/MeRaazi/auto-uninstaller/main/.github/screenshot/preview.png)


## ➤ Features

- **Two Powerful Modes:**
  - **Blocklist Mode:** Automatically uninstalls any app that matches a user-defined blocklist.
  - **Aggressive Mode:** Automatically uninstalls *any* newly installed app that is not on the whitelist.
- **Export/Import:** Backup your blocklist, whitelist, and logs.
- **Theme Support:** Switch between Light, Dark, and System default themes.

## ➤ Usage

After installation, you can access the module's web interface to configure its settings.

1.  Open your KernelSU manager app.
2.  Navigate to the "Modules" section and find "Auto-Uninstaller".
3.  Tap on the module to open its web UI.
